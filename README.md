# Welcome to module-android-network!

Network is an library for connect your apis

## Setup

Run the following git commands 

```bash
git submodule init
git submodule update
```

## Usage

```kotlin
dependencies {
    ...
    implementation("com.afoxplus.android:network:1.0.0")
}
```
